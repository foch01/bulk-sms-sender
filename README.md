## Prerequisites:

- A customer database in this form
```json
{
  [{
    "name": "John"
    "tel": "0608678698"
  },
  {
    "name": "Luc"
    "tel": "0608678624"
  }]
}
```


This database can be exposed publicly (for example) just the time of sending SMS. You can also store the data in a private database for more security.
